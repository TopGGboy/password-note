🧪 测试建议

测试步骤：

  1. 启动项目：npm run serve
  2. 登录系统：确保有有效的认证token
  3. 输入主密码：确保加密密钥可用
  4. 进入密码列表：查看现有密码条目
  5. 点击密码条目：进入详情页
  6. 点击编辑按钮：打开编辑模态框
  7. 修改密码信息：测试各个字段的更新
  8. 保存更改：验证API调用和数据更新
